# Machines

C-Star aims to be deployable on a wide range of HPC systems, but so far it has been tested on a limited number.

| Name of system                                                                          | ROMS-MARBL build/run support | Orchestration Support                                                 |
|-----------------------------------------------------------------------------------------|------------------------------|-----------------------------------------------------------------------|
| MacOS ARM64 (Apple Silicon)                                                             | ✅                            | Limited (terminal must stay active, cannot resume terminated session) |
| [RCAC Anvil](https://www.rcac.purdue.edu/anvil)                                         | ✅                            | ✅                                                                     |
| [NERSC Perlmutter](https://docs.nersc.gov/systems/perlmutter/architecture/)             | ✅                            | Untested                                                              |
| [NCAR Derecho](https://ncar-hpc-docs.readthedocs.io/en/latest/compute-systems/derecho/) | ✅                            | Untested                                                              |
| [SDSC Expanse](https://www.sdsc.edu/support/user_guides/expanse.html)                   | ✅                            | Untested	                                                             |
