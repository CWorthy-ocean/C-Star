Blueprint Schemas
#################

Schemas for the built-in blueprints are listed here.


ROMS-MARBL Blueprint
--------------------

The `roms_marbl` blueprint enables the orchestration of an application that 
executes a ROMS simulation.

- :download:`ROMS-MARBL Blueprint Schema v1.0.0 </schemas/bp/roms_marbl/roms_marbl_schema.1.0.0.json>`


Hello World Blueprint
---------------------

This `hello_world` blueprint is an example starting point for creating and
running custom applications with user-provided blueprints.

- :download:`Hello World Blueprint Schema v1.0.0 </schemas/bp/hello_world/hello_world_schema.1.0.0.json>`


Plotter Blueprint
-----------------

This `plotter` blueprint is a slightly more complex example of creating and
running a custom application with user-provided blueprints.


- :download:`Plotter Blueprint Schema v1.0.0 </schemas/bp/plotter/plotter_schema.1.0.0.json>`
